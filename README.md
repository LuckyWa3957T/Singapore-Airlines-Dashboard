# ✈️ Singapore Airlines Dashboard (Power BI)

## 📊 Project Overview

This project presents a Power BI dashboard designed to analyze airline data such as customer behavior, ticket pricing, and overall performance.

---

## 📌 Key Features

* Profitability Analysis
* Customer Count Tracking
* Loyalty Points Insights
* Ticket Price Trends
* Booking Class Comparison
* Route-wise Analysis
* Delay vs Flight Duration Analysis
* Churn Distribution
